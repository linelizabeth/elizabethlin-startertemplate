Refresh

APP, UX/UI

Refresh is an app that uses spontaneity and fun activities to help users mitigate their frustration with daily life in quarantine. By using an algorithm that generates a unique feed for the individual, the user will be able to be consistently entertained by the different choices they are presented with.
